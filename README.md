# Arya Amoriya — Portfolio

Personal developer portfolio for **Arya Amoriya**, a Full-Stack Developer and Computer Science Engineering (AI-ML) student, showcasing full-stack and AI-integrated project work.

## About

This site is a single-page portfolio built to highlight full-stack development work across Next.js, React, and MERN, with a focus on projects that integrate generative AI (Google Gemini API) into real, deployed products.

## Featured Work

- **CareerPilot AI** — Flagship project. An AI-powered career-intelligence platform where users upload a resume and receive Gemini-generated ATS scoring, strengths/weaknesses, skill-gap analysis, salary predictions, and a learning roadmap.
- **HistoryCast AI** — AI-moderated debates between historical figures, built on Next.js 15 with Prisma and PostgreSQL.
- **AI Startup Validator** — MERN app that scores startup ideas on market potential, scalability, and risk using the Gemini API.
- **Movie Mania, Fits Gallery, Book Villa, Anime Platform** — React-based frontend and full-stack projects covering UI/UX, state management, and REST API integration.

Each project card on the site links to its GitHub repository and live demo where available.

## Tech Stack

Built with plain **HTML, CSS, and JavaScript** — no framework or build step required.

- Fonts: Space Grotesk, IBM Plex Sans, IBM Plex Mono (Google Fonts)
- Scroll-reveal animations via the Intersection Observer API
- Fully responsive layout (mobile, tablet, desktop)
- Deployed via **GitHub Pages**

## Running Locally

No build step needed — just open the file directly:

```bash
git clone https://github.com/Aaryaa-10/arya-portfolio.git
cd arya-portfolio
open index.html   # or double-click the file
```
